**Description:**
A full-stack web application built with Django, Django REST Framework, and Angular. The application implements user authentication using JWT tokens, allowing users to register, log in, access protected pages, and securely manage sessions through a modern frontend connected to a REST API backend.

**Features:**
-  Registration
- User Login & Logout
- JWT Authentication
- User Session Management
- REST API Integration
- Angular Frontend with Django Backend

**Tools Used:**
- Django
- Django REST Framework
- Angular
- TypeScript
- SQLite
- JWT Authentication
